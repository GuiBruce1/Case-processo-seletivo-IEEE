# Case-processo-seletivo-IEEE
Esse repositório é destinado a guardar os materiais e protótipos desenvolvidos durante a realização de um case para entrar no IEEE CEFET-RJ. Fui o líder de minha equipe, sendo responsável por gerir e organizar como, quando e o que iríamos fazer no projeto. Tive como membros os alunos Luciano Cunha, Lucas Torres e Erick Iemini. O case consistia em desenvolvermos um jogo de Damas tradicional utilizando a linguagem C# na Unity engine
